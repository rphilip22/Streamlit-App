## App Description
This app will help users manage their personal finances better by obtaining visuals and insights from their financial data, especially if the user holds multiple bank accounts on multiple platforms.
 
## Intended Users
This app is intended for students interested in managing and obtaining better insights into their personal finances.
 
## App Features
1. Add a transaction (income or expense)
2. View all transactions
3. Show current balance
4. Provide insights into their finances
5. Save data to a file (CSV)
